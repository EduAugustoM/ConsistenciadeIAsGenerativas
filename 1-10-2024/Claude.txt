Claro, vou criar um código em Python para encontrar números primos em um intervalo fornecido. Aqui está o código:



```python
def eh_primo(n):
    if n < 2:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

def encontrar_primos(inicio, fim):
    primos = []
    for num in range(max(2, inicio), fim + 1):
        if eh_primo(num):
            primos.append(num)
    return primos

# Exemplo de uso
inicio = 10
fim = 50
lista_primos = encontrar_primos(inicio, fim)
print(f"Os números primos entre {inicio} e {fim} são: {lista_primos}")

```
